6) Menciona las estucturas repetitivas y escribe la sintaxis de cada una:

1-Ciclo for
i=0 valor inicial
i<=0 número de veces que se repetirá el ciclo
i-- incremento o decremento.
for (i=0; i<=10; i--){
console.log("accion a ejecutar");
}

2-while
éste ciclo se ejecutará mientras se cumpla una condición determinada , es decir , el ciclo no se ejecutará ni una sola vez si inicialmente no se cumple la condición.
while(condicion){
console.log("accion a ejecutar");
}

3-Do while
do{
("accion a ejecutar");
}while(condicion)
en Do While se ejecutará por lo menos una vez el proceso y al ultimo cumprueba si la condición es verdadera.

Los objetos en javascript son similares a los objetos en la vida real , estan formados por propiedades y métodos o acciones

Objeto por literal:
var nombreObjeto{
propiedad:valor,
propiedad:valor,
propiedad:valor,
}
Objeto por prototipo o constructor:
var NombreObjeto=function(propiedades){
this.propiedad:propiedad,
this.propiedad:propiedad,
this.propiedad:propiedad,
}
instancia var objeto=new Objeto();
